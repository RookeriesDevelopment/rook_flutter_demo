# rook_flutter_demo

Demo app for rook_sdk_apple_health, rook_sdk_health_connect and rook_sdk_samsung_health.

## Configuration

In the root folder create a `.env` file with the following properties:

```properties
clientUUID="clientUUID"
packageName="packageName"
bundleId="bundleId"
secret="secret_word"
```

> [!IMPORTANT]
> Don't forget to also register your packageName, bundleId and secret in the ROOK portal.
